# AI Resume Analyser 🤖

An AI-powered resume analyser built with Flask and Groq (Llama 3).

## Features
- Upload PDF or TXT resume
- Get Overall, ATS, Skills & Job Match scores
- Skills extraction
- ATS compatibility check
- AI Chat about your resume

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Add your Groq API key in app.py

3. Run the app:
python app.py

4. Open browser at http://127.0.0.1:5000
